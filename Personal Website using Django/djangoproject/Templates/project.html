Flight Fare Prediction Model (Machine Learning):

In this project, I worked on the dataset of the airplane which contains various data such as Flight Company, Source Airport, Destination Airport, Air Fare, Number of Stops, etc. I did Data cleaning and Data pre-processing on the raw data and converted all the data in such a format which can be easily understandable by different algorithms. Then I split the dataset into Train and Test data. I applied 5 different algorithms to this data set for prediction purposes and I got a maximum accuracy of 84% in the Random Forest Algorithm. To increase it further I hyper-tuned the model by using RandomizedSearchCV and at last, I got a final accuracy of 86.6%.
